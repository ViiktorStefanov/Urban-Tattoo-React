# HOW TO START APPLICATION:

Type in the terminal
```
npm i
```
Install all dependecies and then go for:

```
npm run start
```

Open new terminal and type:
```
cd ./server
```
Install all dependecies for the server: 
```
npm i
```
Then run the server:

```
npm run server
```
